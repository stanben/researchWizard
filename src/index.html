sourceLink is a tool used to view and evaluate sources of a person and his immediate family members.

Within the navagation header at the top of the web page is an information box which provides hints as what to do as well as an information summary when an action is performed. At times sourceLink will shorten words for names and places in order to fit the space available within the web page. When this word is selected, the information box will show the entire word.

The person area is set to a background color of blue if the reference person is male, pink if the reference person is female, or grey if the reference person's gender is unknown. The name and vital information associated with the reference person is at the top of the person area. Below the reference person and to the left is his spouse(s) and children of that spouse. Below the reference person and to the right are his parents and siblings. Each piece of vital information in the person area is highlighted with a red box outline when the mouse points at it. Clicking on a piece of vital information will select it which is displayed with red fill. When selected, all of the sources which confirm this information are highlited. if the ctrl key is pressed while selecting a piece of vital information, then the person associated with this information becomes the reference person and all areas are updated to reflect the information associated with this change.

Within the region set to a background color of green: attached sources are placed in a chronological order from birth to death.

Within the region set to a background color of tan: possible sources are also placed in chronological order from birth to death.
